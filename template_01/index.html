Чому ми це вчимо?

Тут пояснення і приклади де конкретно це використовується в додатках та іграх.

Приклад HTML коду

Тут пояснення матеріалу. Так виглядають акценти. 

<!-- приклад HTML коду -->
<h1>Hello, world!</h1>

Так виглядають картинки (мають максимальну висоту 350px). Зазвичай тут має бути скріншот з результатом коду, або ілюстрація/схема для пояснення теми:

Приклад CSS коду

Для правильної підсвітки коду бажано додавати до тегу code класи: language-html, language-css або language-javascript. 

/* приклад CSS коду */
* { 
  box-sizing: border-box; 
}

Приклад JS коду

Якщо дадати до тегу code клас nocopy, код не можна буде виділити і скопіювати (щоб учні переписували його самі).

// приклад JS коду
console.log("Hello, world!");

Важливо: Так можна наголосити на важливій інформації до цієї частини теми.

Примітка: Так можна написати додаткову інформацію до цієї частини теми.
