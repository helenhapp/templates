Приклад довгого коду

const viewDataButton = document.getElementById("viewData");
const inputArea = document.getElementById("inputArea");
const outputDiv = document.getElementById("output");

function run() {
  const jsonData = inputArea.value;
  try {
    const data = JSON.parse(jsonData);

    outputDiv.innerHTML = "";

    const pageTitle = document.createElement("h2");
    pageTitle.textContent = `Page of ${data.name} ${data.surname}`;
    outputDiv.appendChild(pageTitle);

    const nameLine = document.createElement("p");
    nameLine.textContent = `Name: ${data.name}`;
    outputDiv.appendChild(nameLine);

    const surnameLine = document.createElement("p");
    surnameLine.textContent = `Surname: ${data.surname}`;
    outputDiv.appendChild(surnameLine);
  } catch (error) {
    outputDiv.textContent = "Invalid JSON input. Please enter valid JSON data.";
  }
}

▶

Завдання

Відтвори сайт як на картинці.

⚡️ Важливо: Щоб отримати підказку, який тег потрібно використати, наведи курсор на потрібний елемент на зображенні.

✨ Примітка: Щоб розмістити кожен блок про нового персонажа з нового рядка (очистивши попередні align), використай тег <br clear="all">.

💡 Підказка: у якому порядку додавати теги?
  1. Головний заголовок сторінки <h1>.
  2. Рядок меню навігації <p> (а всередині нього — чотири посилання <a>).
  3. Блок 1 (Маріо): Спочатку картинка <img align="left">, потім ім'я героя <h2> та текст опису <p>.
  4. Очищення обтікання <br clear="all">.
  5. Блок 2 (Луїджі): Картинка <img align="right">, потім його ім'я <h2> та опис <p>.
  6. Знову <br clear="all"> і так само продовжуй для кожного наступного героя (чергуючи left та right для картинок).
  7. В самому кінці сторінки — посилання "Нагору".
