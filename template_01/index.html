Приклад CSS коду

Для правильної підсвітки коду бажано додавати до тегу code класи: language-html, language-css або language-javascript. 

/* приклад CSS коду */
* { 
  box-sizing: border-box; 
}

⚡️ Важливо: Так можна наголосити на важливій інформації.

✨ Примітка: Так можна написати додаткову інформацію.

Приклад JS коду

Якщо дадати до тегу code клас nocopy, код не можна буде виділити і скопіювати (щоб учні переписували його самі). 

// приклад JS коду (який не можна копіювати)
console.log("Hello, world!");
Hello, world!
Тут текст з прикладу, який можна скопіювати.
